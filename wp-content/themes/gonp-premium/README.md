# GoNP Premium Homepage Theme

Custom WordPress theme focused on a premium, conversion-first homepage for GaysOfNationalParks.com.

## What this includes
- Custom `front-page.php` with the exact section order requested.
- Sticky transparent navigation with blur-on-scroll behavior.
- Mobile-first responsive layout and swipe-friendly featured trips carousel.
- Subtle reveal animations and lightweight JS interactions.
- ACF field group JSON with editable fields, repeaters, and section visibility toggles.

## Setup
1. Install this theme in `wp-content/themes/gonp-premium` (zip the folder contents including `style.css`, `index.php`, `templates/index.html`, `header.php`, and `footer.php`).
2. Activate the theme in WordPress admin.
3. Set a static front page under **Settings → Reading**.
4. Install and activate **Advanced Custom Fields Pro** (required for repeaters).
5. Visit the front page editor and fill the `GoNP Homepage` field group.

## Editable groups
- Hero (video/poster/headline/CTAs/nav)
- Quick clarity band
- Featured trips (repeater)
- Belonging section
- Trust signals (stats/logos/testimonials repeaters)
- Mission section (support cards repeater)
- Pathways (repeater)
- Community portraits (repeater)
- Stay connected section
- Footer brand + tagline

## Notes
- Theme ships with placeholder defaults so the page is renderable before data entry.
- Replace placeholders with approved copy/media before production launch.
